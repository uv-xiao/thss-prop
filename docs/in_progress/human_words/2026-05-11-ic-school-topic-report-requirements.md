# 2026-05-11 集成电路学院博士选题报告要求

## 用户原文

> 转发更多关于这个报告的要求：内容需要包括但不限于：研究目的和选题意义、关键科学问题、创新性、研究方法、
> 研究内容与阶段性成果、研究计划、参考文献。博士选题报告为集成电路学院博士研究生培养中最重要的环节之一，是博士研究生提高学位论文质量的重要保障。在博士学位论文开始前或初始阶段，博士研究生应在本门学科广泛调查研究、阅读文献资料、搞清楚主攻方向上的前沿成果和发展动态的基础上，提出学位论文选题。选题应尽可能对学术发展、经济建设和社会进步有重要意义。按照培养计划制定的时间，就选题意义、前人相关成果、拟解决的关键科学问题及创新性、拟采取的研究方案等方面作学位论文选题报告，考核委员会对选题的先进性、创新性、研究方案的可行性以及初期研究进展进行深入评估，重点考察学生是否具有完成创新性科学研究的能力和条件，并提供指导建议。
>
> 本选题报告实施办法适用于微电子学与固体电子学、集成电路科学与工程专业。
>
>
> 1.考核内容及形式：学生围绕相关研究课题，提交书面报告，然后做口头报告和答辩。重点考查论文选题、文献综述、创新性、研究方案、基础理论和专门知识、科研能力、学术规范以及对研究课题的认识和理解。
>
> 2.考核委员会：主要由各学科的至少五名专家（硕导或博导，其中至少三名博导）组成，书面报告评审和答辩均采用导师参加制度，导师不能担任组长。
>
> 3.书面报告：以学科研究目的和选题意义、关键科学问题、创新性、研究方法、研究内容与阶段性成果、研究计划为主要内容，撰写选题书面报告。
>
> 4.答辩考核：以口头报告方式进行答辩考核。报告内容须涵盖选题意义、研究进展和创新性及实用性价值等部分。考核委员会还可以对更广泛的内容提问，对相关领域基础知识、文献调研与拓展、科研思维能力、报告的逻辑性以及语言表达能力等进行考察。

## Agent Interpretation

- 这是制度性要求，应提升到强制报告规则，而不是普通偏好。
- 后续全文审阅必须检查报告是否覆盖：研究目的和选题意义、关键科学问题、创新性、研究方法、研究内容与阶段性成果、研究计划、参考文献。
- 引言和综述不仅要讲背景，还要证明已经广泛调研本学科前沿成果和发展动态。
- 报告需要显式支撑答辩考核：选题先进性、创新性、方案可行性、初期研究进展、科研能力与条件、学术规范、逻辑性和表达能力。
- 报告主题所在学科语境应按微电子学与固体电子学、集成电路科学与工程理解。
