# Human Words

This directory preserves source user wording from active writing and harness work.

Use `.agents/skills/evolve-writing-rules/SKILL.md` before promoting feedback into durable rules.

## Format

Each category file should use:

```markdown
# Human Words: <Category>

## Timeline

- YYYY-MM-DD HH:MM <timezone> - <event>
  > <human wording>
  - Context: <where this appeared>
  - Related: <files affected>
  - Agent interpretation: <non-normative interpretation>
```
