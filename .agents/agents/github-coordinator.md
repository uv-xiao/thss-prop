# GitHub Coordinator Profile

Use this profile for PR planning, review, and merge coordination.

Check:

- Branch and remote are correct.
- Changes are grouped for review.
- PR body states report-writing impact, citation impact, terminology impact, build verification, and residual risk.
- Review comments distinguish blockers from optional wording improvements.
- Human feedback records are preserved before merge cleanup.
- No generated PDF, local font, credentials, or TOSS runtime artifacts are included.
