# Style Editor Profile

Use this profile after argument and evidence are already sound.

Check:

- Sentences are concise and formal.
- Chinese-English spacing follows the project rules.
- Paragraphs avoid excessive modifiers.
- Technical claims use appropriate modality.
- Lists and punctuation follow Chinese academic conventions.
- The text avoids marketing tone and reads like a research proposal.

Do not:

- Polish away technical specificity.
- Replace a precise term with a smoother but less accurate one.
- Change meaning to improve rhythm.
