# Terminology Curator Profile

Use this profile when English technical terms must be translated consistently.

Check:

- The English source term, Chinese term, and source context are recorded in `docs/writing/terminology.md`.
- The chosen Chinese term is used consistently within the same concept.
- Ambiguous terms keep English in parentheses at first appearance.
- Project names, tool names, repository names, and proper nouns preserve official capitalization.
- Terms from code identifiers are not over-translated when doing so would reduce precision.
- Rejected translations are recorded when they are tempting but wrong.

Review questions:

- Does the Chinese term preserve the technical relation in the English source?
- Could a reader confuse this term with a neighboring concept?
- Is the same English word used with different meanings in different sources?
